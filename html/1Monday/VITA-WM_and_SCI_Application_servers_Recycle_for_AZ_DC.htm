Sample Mail Request

 

Step 1. Bring down MIP, MDA and WM applications in both nodes.

https://lin-ash-wmspsci.vitshoppe.com:3443/sd3/index.html (system/*****)

Select PROD-AZ-NODE-1 & PROD-AZ-NODE-2 cluster and stop the applications.

Verify the Status in Action Status Tab.

Step 2. Login to the server and check if all the processes are stopped .


ps -ef |grep /apps/scope/mip | grep -v grep
ps -ef |grep /apps/scope/mda | grep -v grep
ps -ef | grep /apps/scope/wm | grep -v grep

Step 3. Check for active RF sessions and kill them if there are any active sessions.

 

ps -ef |grep rfclient | grep -v grep

Step 4. Start MIP in Node 1 and Node 2. Verify logs for error/exceptions

 

tail -f /apps/scope/mip/profile-root/wc-mip/logs/mip-main.log

Step 5. Start MDA in Node 1 and Node 2. Verify logs for error/exceptions

 

tail -f /apps/scope/mda/profile-root/mda/log/mda-main.log

 

Step 6. Start WM in Node 1 and Node 2. Verify logs for error/exceptions.

 

tail -f /apps/scope/wm/profile-root/wm/log/wm-main.log

 

 

Step 7. Stop SCI and kill if any stale process is running.

Log into sci server(lin-ash-wmspsci.vitshoppe.com) and go to /etc/init.d

Execute ./sciadm stop . This will stop cognos application

Check if there any loner process and kill them . ps -ef | grep sciadmin | grep cognos | grep -v grep | grep -v bash

Step 8. Start SCI application and verify the logs.

/apps/scope/sci/cognos/c10_64/logs/cogserver.log
